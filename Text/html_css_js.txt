DOM → Focuses on the document structure (HTML/XML), enabling manipulation of elements, attributes, and styles.

BOM → Focuses on the browser environment, enabling control over windows, navigation, history, and screen information.

<script> Tag - Used to include JavaScript code (either inline or from an external file).
	Attributes:
	src → Path to external JavaScript file.
	type → MIME type (default is text/javascript).
	defer → Executes script after HTML parsing.
	async → Executes script asynchronously.

	<script src="script.js" defer></script>

	<!-- Inline JavaScript -->
	<script>
    		console.log("Hello from inlineJavaScript!");
	</script>


<link> Tag - Used to link external resources like CSS stylesheets, icons, etc.
	Attributes:
	rel → Relationship of the linked file (e.g., stylesheet, icon).
	href → Path to the resource.
	type → MIME type (optional for CSS).

	<!-- External CSS -->
	<link rel="stylesheet" href="styles.css">
