	. Not possible for static methods because overriding requires runtime polymorphism (dynamic binding),

Wrapper classes -> methods
	1. valueOf()
		Used to convert a primitive or String into a wrapper object.
		Example:
		Integer i = Integer.valueOf("10");
		Double d = Double.valueOf(20.5);
	2. parseXxx()
		Converts a String into a primitive value.
		Example:
		int n = Integer.parseInt("100");
		double x = Double.parseDouble("45.6");
		boolean b = Boolean.parseBoolean("true");
	3. xxxValue()
		Converts a wrapper object back to its primitive.
		Example:
		Integer i = 50;
		int num = i.intValue();
		double val = i.doubleValue();
	4. compareTo()
		Compares two wrapper objects and returns -1, 0, or 1.
		Example:
		Integer a = 5, b = 10;
		a.compareTo(b);  // returns -1
	5. equals()
		Checks if two wrapper objects have equal values.
		Example:
		Integer x = 20, y = 20;
		x.equals(y);  // true
	6. toString()
		Converts a wrapper object to a String.
		Example:
		Integer n = 25;
		String s = n.toString();  // "25"
